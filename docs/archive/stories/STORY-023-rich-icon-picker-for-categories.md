# STORY-023: Rich Icon Picker for Categories

## Status

## Epic

## Depends on

## Description
V1 of category creation defaults new categories to the `tag` icon. This story adds a searchable icon picker from Obsidian's icon set, allowing users to choose meaningful icons for their custom categories.

## Acceptance Criteria
- [ ] Icon picker component with search/filter over Obsidian icon set
- [ ] Available in both "+" modal and settings management UI
- [ ] Selected icon previewed before confirmation
